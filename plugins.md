## 使用方法

在 Claude Code 中使用 `/plugin` 命令管理插件：

- `Discover`: 发现新插件
- `Installed`: 查看已安装插件
- `Marketplaces`: 管理插件市场
- `Errors`: 查看错误信息

快捷键：
- `Space`: 切换选中
- `Enter`: 查看详情
- `Delete`: 卸载插件
- `Esc`: 返回
- `Tab`: 切换标签页
