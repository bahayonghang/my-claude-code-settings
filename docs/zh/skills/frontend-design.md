# frontend-design

创建独特的生产级前端界面，具有高设计质量。

## 理念

此技能创建避免通用"AI 风格"美学的界面。每个设计应该是：

- **独特** - 令人难忘且独一无二
- **生产级** - 功能完善且精致
- **有意图** - 清晰的美学方向

## 设计思维

编码前，确定一个大胆的美学方向：

1. **目的**：解决什么问题？谁在使用？
2. **基调**：选择一个极端的美学风格
3. **约束**：技术要求
4. **差异化**：什么让它令人难忘？

## 美学方向

- 极简主义
- 极繁主义
- 复古未来
- 有机/自然
- 奢华/精致
- 俏皮/玩具感
- 编辑/杂志风
- 粗野主义
- 装饰艺术/几何
- 柔和/粉彩
- 工业/实用

## 关注领域

### 排版
- 选择独特字体（避免 Arial、Inter、Roboto）
- 展示字体搭配精致正文字体
- 意想不到的、有个性的选择

### 颜色与主题
- 坚持统一的美学
- 使用 CSS 变量保持一致性
- 主色调配合鲜明的强调色

### 动效
- 高影响力动画
- 页面加载时的交错显示
- 令人惊喜的悬停状态
- 尽可能使用纯 CSS

### 空间构图
- 意想不到的布局
- 不对称和重叠
- 打破网格的元素
- 大量留白或控制密度

### 视觉细节
- 渐变网格
- 噪点纹理
- 几何图案
- 分层透明度
- 戏剧性阴影
- 自定义光标

## 反模式

永远不要使用：
- 过度使用的字体（Inter、Roboto、Arial）
- 陈词滥调的配色（白底紫色渐变）
- 可预测的布局
- 千篇一律的设计

## 输出

可工作的代码（HTML/CSS/JS、React、Vue 等），具有：
- 生产级且功能完善
- 视觉冲击力强且令人难忘
- 统一且有清晰美学观点
