# excalidraw

生成手绘风格图表（Excalidraw JSON 格式）。

## 使用场景

- 架构图
- 流程图
- 系统设计图
- 时序图

## 关键规则

1. **箭头绑定**：箭头必须双向绑定
   - 箭头需要 `startBinding` 和 `endBinding`
   - 矩形需要 `boundElements` 数组

2. **文字尺寸**：文字元素必须有 `width` 和 `height`

3. **背景区域**：必须完全覆盖包含的元素，四周留 40px 内边距

4. **无重叠**：箭头不能穿过无关组件

## 颜色系统

| 用途 | 背景 | 描边 |
|------|------|------|
| 主要 / 阶段 1 | `#a5d8ff` | `#1971c2` |
| 次要 / 阶段 2 | `#b2f2bb` | `#2f9e44` |
| 强调 / 共享 | `#fff3bf` | `#e67700` |
| 存储 / 状态 | `#d0bfff` | `#7048e8` |

## 布局规则

- 坐标对齐到 20 的倍数
- 组件间距：100-150px
- 标准组件大小：140×60
- 背景区域：透明度 30

## 避免箭头交叉

当箭头会穿过组件时：

1. **三角形布局**：3 个组件带返回箭头
2. **菱形布局**：4 个组件带返回箭头
3. **绕行路径**：箭头从组件上方/下方绕行

## 输出

- **格式**：`.excalidraw.json`
- **使用**：拖入 excalidraw.com 或用 VS Code 扩展打开
