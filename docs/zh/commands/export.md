# 导出会话上下文 (Export)

`/export` 命令用于分析当前会话历史，并生成结构化的总结，以便于上下文继承。

## 用法

```bash
/export
```

## 描述

此命令会对当前对话和代码库状态进行全面审查，生成一份 **交接总结 (Handoff Summary)**。该总结将保存为 Markdown 文件（例如 `session_context.md`），可以将其输入到新的会话中，从而无缝恢复上下文。

总结内容包括：
- 🎯 **会话目标**：本次会话的主要目标是什么？
- ✅ **成就**：已完成的任务和已实现的功能。
- 🧠 **关键决策**：架构选择和重要发现。
- 🚧 **下一步计划**：下一次会话的可执行事项。
- 📂 **关键文件**：被修改或分析的相关文件列表。

## 平台支持

| 平台 | 支持情况 | 文件格式 |
|----------|---------|-------------|
| Claude Code | ✅ | Markdown (`.md`) |
| Gemini CLI | ✅ | TOML (`.toml`) |
| Codex CLI | ❌ | 暂不支持 |

## 输出示例

```markdown
# 会话交接: [日期]

## 🎯 目标
使用 OAuth2 实现用户登录流程。

## ✅ 成就
- 创建了包含 `login` 函数的 `auth_service.py`。
- 更新了 `requirements.txt` 以包含 `python-jose`。

## 🚧 下一步计划
- 实现令牌刷新逻辑。
- 为 `auth_service.py` 添加单元测试。
```
