# 中文学术写作规范

## 一、口语化表达纠正

### 1.1 程度副词替换
| 口语 | 学术 |
|------|------|
| 很多 | 大量、众多、若干 |
| 非常 | 极为、显著、相当 |
| 特别 | 尤其、尤为、格外 |
| 一些 | 部分、若干、某些 |
| 很好 | 优异、显著、卓越 |

### 1.2 主语替换
| 避免 | 使用 |
|------|------|
| 我们 | 本文、本研究、作者 |
| 我 | （尽量避免第一人称单数）|
| 你们 | （不应出现）|

### 1.3 动词规范化
| 口语 | 学术 |
|------|------|
| 用 | 采用、使用、运用、利用 |
| 做 | 进行、开展、实施、执行 |
| 看 | 观察、分析、研究、考察 |
| 想 | 认为、推测、假设、考虑 |
| 试 | 尝试、探索、验证 |

## 二、绝对化词汇规避

### 2.1 禁用词汇
- ❌ 显然、毫无疑问、众所周知
- ❌ 必然、绝对、完全、最好、最优
- ❌ 肯定、一定、当然、无疑

### 2.2 替代表达
| 绝对化 | 学术化 |
|--------|--------|
| 显然 | 研究表明、实验结果显示 |
| 毫无疑问 | 可以认为、有理由相信 |
| 众所周知 | 已有研究指出、文献表明 |
| 必然 | 往往、通常、一般而言 |
| 最好 | 较优、更优、具有优势 |

## 三、逻辑连接词

### 3.1 递进关系
- 此外、另外、进一步、更为重要的是
- 不仅...而且...、既...又...

### 3.2 转折关系
- 然而、但是、不过、尽管如此
- 与此相反、相比之下

### 3.3 因果关系
- 因此、由此可见、综上所述
- 鉴于此、基于上述分析

### 3.4 举例说明
- 例如、譬如、以...为例
- 具体而言、特别是

## 四、常见语病

### 4.1 搭配不当
| 错误 | 正确 |
|------|------|
| 发挥问题 | 发现问题 |
| 增加效率 | 提高效率 |
| 扩大精度 | 提高精度 |
| 改进缺点 | 改正缺点 |

### 4.2 成分残缺
❌ "通过实验，验证了方法的有效性。"（缺主语）
✅ "通过实验，本文验证了该方法的有效性。"

### 4.3 语序不当
❌ "该方法不仅提高了效率，而且也降低了成本。"
✅ "该方法不仅降低了成本，而且提高了效率。"（先易后难）

## 五、标点符号规范

### 5.1 中文标点
- 使用中文标点：，。！？；：""''（）【】
- 中文句子末尾使用中文句号"。"

### 5.2 英文标点
- 英文术语后可用英文标点
- 公式后的说明用英文逗号

### 5.3 混用规则
- 中文语境中的英文单词后用中文标点
- 括号内全英文时用英文标点

## 六、数字与单位

### 6.1 数字使用
- 精确数值：阿拉伯数字（1, 2, 3）
- 概数：汉字（几十、数百）
- 序数：第一、第二（非 1st, 2nd）

### 6.2 单位规范
- 国际单位制（SI）
- 单位与数值之间空一格
- 使用正体字母（非斜体）
