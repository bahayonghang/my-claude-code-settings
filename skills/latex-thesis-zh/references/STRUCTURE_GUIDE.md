# 学位论文结构指南

## 一、论文整体结构

### 前置部分（必需）
1. **封面** (cover)
   - 论文题目（中英文）
   - 作者姓名
   - 导师姓名
   - 学科专业
   - 提交日期

2. **原创性声明和使用授权** (declaration)
   - 原创性声明
   - 使用授权书
   - 签名页

3. **摘要** (abstract)
   - 中文摘要（500-1000字）
   - 关键词（3-5个）
   - 英文摘要 (Abstract)
   - Keywords

4. **目录** (toc)
   - 自动生成
   - 包含章节编号和页码

5. **符号对照表**（推荐）
   - 符号说明
   - 缩略语表

### 正文部分

6. **第一章：绪论**
   - 研究背景
   - 研究意义
   - 研究问题/科学问题
   - 主要贡献
   - 论文结构

7. **第二章：相关工作/文献综述**
   - 研究现状分类综述
   - 现有方法对比分析
   - 研究空白识别

8. **第三章至第N-1章：核心内容**
   - 方法/模型/算法
   - 实验设计
   - 结果分析

9. **第N章：总结与展望**
   - 工作总结
   - 研究局限
   - 未来方向

### 后置部分（必需）

10. **参考文献** (bibliography)
    - GB/T 7714 格式
    - 按引用顺序编号

11. **致谢** (acknowledgment)
    - 感谢导师
    - 感谢合作者
    - 感谢资助机构

12. **攻读学位期间发表论文目录** (resume/publications)
    - 已发表论文
    - 已接收论文
    - 在投论文（可选）

13. **附录**（如有）
    - 补充材料
    - 代码清单
    - 详细推导

## 二、章节编号规范

### 层级结构
```
第1章 绪论
  1.1 研究背景
    1.1.1 问题描述
    1.1.2 挑战分析
  1.2 研究意义
第2章 相关工作
  2.1 ...
```

### 编号格式
- 章：第X章（中文数字或阿拉伯数字）
- 节：X.1
- 小节：X.1.1
- 段：X.1.1.1（尽量避免超过四级）

## 三、页面设置

### 页边距
| 位置 | 尺寸 |
|------|------|
| 上 | 3.0 cm |
| 下 | 2.5 cm |
| 左 | 2.5-3.0 cm |
| 右 | 2.5 cm |

### 页眉页脚
- 页眉：章标题或"XX大学博士学位论文"
- 页脚：页码（居中）
- 摘要页使用罗马数字
- 正文使用阿拉伯数字

### 字体字号
| 内容 | 字体 | 字号 |
|------|------|------|
| 正文 | 宋体 | 小四/五号 |
| 章标题 | 黑体 | 三号 |
| 节标题 | 黑体 | 四号 |
| 表题/图题 | 宋体 | 五号 |

## 四、常见检查清单

### 前置部分
- [ ] 封面信息完整准确
- [ ] 声明页已签名
- [ ] 摘要字数符合要求
- [ ] 关键词数量合适
- [ ] 目录页码正确
- [ ] 符号表完整

### 正文部分
- [ ] 第一章包含研究背景、意义、贡献
- [ ] 文献综述覆盖主要相关工作
- [ ] 方法描述清晰完整
- [ ] 实验设计合理
- [ ] 结果分析客观
- [ ] 最后一章有总结和展望

### 后置部分
- [ ] 参考文献格式统一
- [ ] 致谢内容得体
- [ ] 论文发表清单完整
