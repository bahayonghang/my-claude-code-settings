# GB/T 7714-2015 参考文献格式规范

## 一、文献类型标识

| 类型 | 标识 | 示例 |
|------|------|------|
| 普通图书 | M | [1] 作者. 书名[M]. 出版地: 出版社, 年份. |
| 期刊文章 | J | [2] 作者. 题名[J]. 刊名, 年, 卷(期): 页码. |
| 学位论文 | D | [3] 作者. 题名[D]. 城市: 学校, 年份. |
| 会议论文 | C | [4] 作者. 题名[C]//会议名. 城市, 年: 页码. |
| 专利 | P | [5] 发明人. 专利名[P]. 国别: 专利号, 日期. |
| 电子文献 | EB/OL | [6] 作者. 题名[EB/OL]. (发布日期)[引用日期]. URL. |
| 标准 | S | [7] 标准名[S]. 标准号, 年份. |
| 报告 | R | [8] 作者. 报告名[R]. 机构, 年份. |

## 二、著录格式示例

### 2.1 期刊文章
```bibtex
@article{example_journal,
  author = {张三 and 李四 and 王五},
  title = {深度学习在图像分类中的应用研究},
  journal = {计算机学报},
  year = {2023},
  volume = {46},
  number = {1},
  pages = {1--15},
  doi = {10.11897/SP.J.1016.2023.00001},
}
```

### 2.2 会议论文
```bibtex
@inproceedings{example_conf,
  author = {Smith, John and Johnson, Mary},
  title = {A Novel Approach to Object Detection},
  booktitle = {Proceedings of CVPR},
  year = {2023},
  pages = {1234--1243},
  address = {Vancouver, Canada},
}
```

### 2.3 学位论文
```bibtex
@phdthesis{example_thesis,
  author = {张三},
  title = {基于深度学习的图像识别算法研究},
  school = {清华大学},
  year = {2023},
  address = {北京},
}
```

## 三、BibLaTeX 配置

### 3.1 使用国标样式
```latex
\usepackage[backend=biber,style=gb7714-2015]{biblatex}
\addbibresource{refs.bib}

% 文档末尾
\printbibliography[title=参考文献]
```

### 3.2 BibTeX 样式
```latex
\bibliographystyle{gbt7714-numerical}  % 数字编号
% 或
\bibliographystyle{gbt7714-author-year}  % 作者-年份

\bibliography{refs}
```

## 四、常见问题

### 4.1 作者姓名格式
- 中文：姓在前，名在后，不加标点
- 英文：姓, 名缩写. 或 姓, 名

### 4.2 多作者处理
- 3人及以下：全部列出
- 4人及以上：前3人 + "等" 或 "et al."

### 4.3 DOI 要求
- 推荐添加 DOI（有 DOI 时必须著录）
- 格式：`doi = {10.xxxx/xxxxx}`

### 4.4 页码格式
- 使用连字符：1--15（双横线）
- 不使用波浪线或单横线

## 五、图表编号规范

### 5.1 图题格式
- 位置：图下方
- 格式："图 3-1 图题内容" 或 "图3.1 图题内容"
- 字体：宋体五号

### 5.2 表题格式
- 位置：表上方
- 格式："表 3-1 表题内容" 或 "表3.1 表题内容"
- 字体：宋体五号

### 5.3 公式编号
- 位置：公式右侧
- 格式：(3.1) 表示第3章第1个公式
- 对齐：右对齐

## 六、章节标题规范

| 级别 | 字体 | 字号 | 对齐 |
|------|------|------|------|
| 章标题 | 黑体 | 三号 | 居中 |
| 节标题 | 黑体 | 四号 | 左对齐 |
| 小节标题 | 黑体 | 小四 | 左对齐 |
| 段落标题 | 黑体 | 五号 | 左对齐 |
